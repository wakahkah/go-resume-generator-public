# Go-Resume-Generator

Get data from json file and using [go-wkhtmltopdf](https://github.com/SebastiaanKlippert/go-wkhtmltopdf) to generate pdf from html template

## Input json file
```
source/resume-data.json
```

## Output pdf file
```
source/resume-data.json
```

## HTML template
```
templates/resume.html
```
